# Project 017: Calculate the Average of a List

## Description
This project asks the user to enter multiple numbers, then calculates their average.  
It teaches beginners about lists, loops, and basic math operations.

## Skills Learned
- Lists  
- Loops  
- Arithmetic operations  
- User input  

## How to Run
1. Open `average_list.ipynb` in Jupyter Notebook.
2. Run all cells.
3. Enter numbers separated by commas.
